Get going fast

Up and running in 3 steps

No build tools, no config files — just Claude Code slash commands. 

# 1. Register the marketplace (once, globally)
/plugin marketplace add kud/claude-plugins

# 2. Install plugins
/plugin install trakt@kud --scope user       # personal tools
/plugin install jenkins@kud --scope project  # shared team tools

# 3. Use a skill
/trakt-whats-on
→ Your watchlist, recently watched, and upcoming releases

/ci-diagnose my-service
→ Root-cause analysis of the latest failing build

/bookmark-search react hooks
→ Top 10 matching bookmarks from your Raindrop library
Marketplace

All 7 plugins

Personal tools install with --scope user, team tools with --scope project.

🤖 opencode Query any opencode-supported model from inside Claude — get a second opinion without leaving your session.
🐙 github-copilot Query GitHub Copilot models via the official SDK — no token setup, uses your existing logged-in credentials.
🏗️ jenkins Full Jenkins control — inspect builds, stream logs, trigger pipelines, manage queues, and diagnose CI failures.
🚩 harness-fme Inspect and control Harness FME feature flags — list environments, audit targeting rules, kill or restore flags.
🔖 raindrop-io Manage your Raindrop.io bookmarks — search your library, save new links, organise into collections, and tag.
📝 google-keep Read and write Google Keep notes — capture thoughts, manage checklists, and search by label, colour, or text.
🎬 trakt Track what you're watching — check in, browse history, manage your watchlist, and get recommendations.

All 11 skills

Every plugin ships with ready-made slash command workflows — no prompt engineering required.

/trakt-checkin <title> Check in to a movie or episode you're about to watch
